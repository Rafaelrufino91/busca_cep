# 🔎 Consulta de Endereço por CEP

Este projeto foi desenvolvido com o objetivo de **praticar Python**, trabalhando com:

- consumo de APIs
- criação de interfaces gráficas com Tkinter
- manipulação de dados em Python

A aplicação permite consultar um endereço a partir de um **CEP**, utilizando a **API ViaCEP** para obter as informações.

---

## 🚀 Tecnologias utilizadas

- Python
- Tkinter
- Requests
- API ViaCEP

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/Rafaelrufino91/busca_cep.git
````

Entre na pasta do projeto:

```bash
cd buscacep
```

Instale a dependência necessária:

```bash
pip install requests
```

---

## ▶️ Como executar

Execute o programa com:

```bash
python buscador.py
```

Uma janela será aberta permitindo consultar um CEP.

---

## 🧪 Funcionamento

1. Digite o CEP desejado
2. Clique em **Pesquisar**
3. O sistema consulta a **API ViaCEP**
4. O endereço correspondente será exibido na tela

Caso o CEP seja inválido ou inexistente, uma mensagem de erro será exibida.
